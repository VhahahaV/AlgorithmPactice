# 托帕的股市分析

## 题目背景

行情扑朔迷离……

望长放短，就是成功的秘诀！

## 题目描述

作为星际和平公司战略投资部的资深员工，托帕总监最擅长的就是股市分析。

托帕的助理为托帕送来了一份股票的每日报价（数组$A$），托帕想要知道某支股票是否可以卖出，就要分析这支股票价格的**跨度**。

当日股票价格的**跨度**被定义为股票价格小于或等于今天价格的最大连续日数（从今天开始往回数，包括今天）。
请你为托帕解出每一天股票的**跨度**（用数组$B$表示，$B[i]$为第i天股票的跨度）。

## 输入格式

第一行输入一个正整数$n$，表示股票每日价格数组$stock$的长度。

第二行输入n个正整数，即$stock$中每个元素。

## 输出格式

输出一行n个正整数，表示股票跨度数组$B$，中间用空格隔开。

# 今天开始我要学会markdown

## 题目背景

_“零分！”_

_“负分！”_

亲爱的拉帝奥博士又把可怜的开拓者的论文退回了，这次竟然是因为开拓者不会写数学公式，竟然只会使用最简单的"+-*/"。智慧的拉帝奥博士觉得开拓者实在是愚不可及，仁慈的拉帝奥博士决定医治开拓者的愚蠢。善良的拉帝奥博士教导开拓者宇宙中最常用的——markdown语法。

## 题目描述

Markdown是一种轻量级标记语言.

借助Markdown中的一些语法，程序可以理解一类简单的数学表达式，并对之求值. 这类简单的数学表达式的集合记为$E$. $\forall A \in E$，$A$的值记为$val(A)$.

$E$中的元素有$3$类，分别称为数（Number，用子集$N$表示）、函数表达式（Function，用子集$F$表示）和运算符连缀式（Concatenation，用子集$C$表示），$\{N,F,C\}$构成$E$的一个划分.

- 1. $N:$ 元素为一个非负整数或一个非负的有限小数对应的表达式，如```10```，```3.1416```等. $\forall A \in N$，$val(A)$即该整数或有限小数的数值，如$val($```10```$)=10$，$val($```3.1416```$)=3.1416$.

- 2. $F:$ 由两种简单的函数递归地定义如下：
    - 1. ```\frac{}{}```函数：$\forall A,B \in E$，若$val(B) \neq 0$，则表达式 $X=$ ```\frac{A}{B}``` $\in F$，且$val(X)=\frac{val(A)}{val(B)}$，如$val($```\frac{2.5}{10}```$)=0.25$.

    - 2. ```\sqrt{}```函数：$\forall A \in E$，若$val(A) \geq 0$，则表达式 $X=$ ```\sqrt{A}``` $\in E$，且$val(X)=\sqrt{val(A)}$，如$val($```\sqrt{\frac{2.5}{10}}```$)=0.5$.

    - 3. ```\sqrt[]{}```函数：$\forall A,B \in E$，若$val(A) \geq 0$，$val(B) > 0$，则表达式 $X=$ ```\sqrt[B]{A}``` $\in E$，且$val(X)=\bigg(val(A)\bigg)^{1 \over val(B)}$，如$val($```\sqrt[3-1]{\frac{2.5}{10}}```$)=0.5$.

- 3. $C:$ $\forall A_1,A_2,\cdots,A_n \in N \cup F$，用$n-1$个运算符将$A_1,A_2,\cdots,A_n$连接成表达式$X$，其中运算符可以为```+```,```-```,```*```和```^```，分别表示一般的加法、减法、乘法和指数运算. 这样的$X$的全体构成集合$C$. $val(X)$的计算服从数学上算术运算的优先级，如$val(A_1+A_2\text{\^ }A_3*A_4)=val(A_1)+val(A_2)^{val(A_3)}*val(A_4)$，更具体地，$val($```2.5+4^0.5*1.5```$)=5.5$.

由定义可以知道，$E$中有许许多多的数学表达式，以下是一些例子：

```\sqrt{\sqrt{2.718}^2}```

```\frac{3.14*2}{5-\sqrt{4}}^0.5```

```1-2*\frac{3}{\sqrt{4+5}}^\frac{\sqrt{6}}{\frac{7}{8}-9}```

## 输入格式

一行，一个字符串$S$，表示待求值的Markdown表达式.

## 输出格式

一行，一个实数$M$（保留$6$位小数）和一个整数$n$，用空格隔开。其中$M$和$n$分别为，$val(S)$的科学记数法表示中的定点数和指数，即$val(S)=M\times10^n$，且$1 \leq |M| < 10$.

## 样例 #1

### 样例输入 #1

```
2.5+4^0.5*1.5
```

### 样例输出 #1

```
5.500000 0
```

## 样例 #2

### 样例输入 #2

```
\sqrt{\sqrt{2.718}^2}
```

### 样例输出 #2

```
1.648636 0
```

## 样例 #3

### 样例输入 #3

```
\frac{3.14*2}{5-\sqrt[1.5*2]{8}}^10
```

### 样例输出 #3

```
1.615786 3
```

## 提示

#### 样例解释
##### 样例 #1
$val(S)=2.5+4^{0.5}\times 1.5=5.5=5.500000\times 10^0$
##### 样例 #2
$val(S)=\sqrt{\big(\sqrt{2.718}\big)^2}\approx 1.648636\times 10^0$
##### 样例 #3
$val(S)=\big(\frac{3.14\times 2}{5-(\sqrt[1.5\times 2]{8})}\big)^{10}\approx 1.615786\times 10^3$

## 数据范围与说明

| 数据点 | $\vert S \vert$ | 性质 |
| :----------: | :----------: | :----------: |
| $1\sim 4$ | $\leq 100$ | 不含```\frac{}{}```,```\sqrt{}```或```\sqrt[]{}```函数 |
| $5,6$ | $\leq 10^5$ | 不含```\sqrt{}```或```\sqrt[]{}```函数 |
| $7,8$ | $\leq 10^5$ | 不含```\frac{}{}```或```\sqrt[]{}```函数 |
| $9\sim 20$ | $\leq 10^5$ | 无 |

对于$100\%$的数据，$|S| \leq 10^5$，且集合$N$中整数和有限小数的数值在$[0,100)$内，且有限小数的小数位数不超过$4$位.

注1：保证表达式合法，即符合题目描述中的定义及各运算符的数学定义，比如不会出现除以$0$或负数的$0.5$次幂的情况.

注2：保证$val(S)$不等于0，即无需考虑输出是```0.000000 0```的情况.

注3：$E$的构造只是借鉴了Markdown的语法，$E$中的表达式未必在Markdown中合法或有相同含义，在Markdown中合法的表达式也未必属于$E$. 比如表达式```2^\sqrt{4}``` $\in E$，但其在Markdown中不合法；表达相同数学含义的```2^{\sqrt{4}}```在Markdown中合法，但不属于$E$.
